# **App Name**: ImageMetaGenius

## Core Features:

- Image Analysis with AI: Leverage AI to analyze image content and extract relevant labels for metadata generation. Will use the specified free AI tools.
- SEO-Optimized Metadata Generation: Automatically generate SEO-optimized titles (8-15 words), descriptions (50-120 words), and keywords (15-25) using extracted labels and AI.
- Metadata Embedding: Embed the generated metadata (title, description, keywords, rating) into image files (JPG, PNG) using EXIF/IPTC fields.
- Batch Processing: Process multiple images (100+) in a batch from a user-specified folder. Shows progress with a progress bar.
- Logging: Create and save a log file (log.txt) with the processing results for each image, including any errors.
- Error Handling: Implement error handling to skip images that cannot be opened or processed by the API, providing informative error messages.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey sophistication and AI focus.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Cyan (#00BCD4) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font for clean readability. 
- Code font: 'Source Code Pro' for any displayed code.
- Simple, minimalist icons to represent file operations and AI actions.
- Subtle animations during file processing and metadata embedding.
